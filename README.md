## Introduction

The YouTube Transcript Scraper allows you to retrieve transcripts from public YouTube videos that have captions enabled. This tool is perfect for developers, content creators, or anyone needing textual content from YouTube videos.

### Key Features
- Fast transcript extraction from YouTube videos.
- Accurate timestamp-based transcript segmentation.
- Ideal for SEO, content repurposing, and research.

## Features

| Feature | Description |
|---------|-------------|
| Fast and Efficient | Retrieves YouTube video transcripts in seconds. |
| Accurate Extraction | Captures full transcript data segmented by timestamps. |
| SEO-Friendly | Transcripts are perfect for SEO optimization and boosting keyword density. |

## What Data This Scraper Extracts

| Field Name  | Field Description |
|-------------|------------------|
| videoUrl    | The input URL of the YouTube video. |
| transcript  | An array containing transcript objects with start time, duration, and text. |

## FAQs

**Q1: Can I use this scraper for private YouTube videos?**
A1: No, the YouTube Transcript Scraper only works with public videos that have transcripts enabled.

**Q2: How do I provide a YouTube URL to the scraper?**
A2: Simply input the full video URL, such as `https://www.youtube.com/watch?v=IELMSD2kdmk`, and the scraper will automatically retrieve the transcript.

**Q3: Does this work with YouTube playlists?**
A3: This actor processes a single video URL at a time. For playlist scraping, additional workflows will be required.
